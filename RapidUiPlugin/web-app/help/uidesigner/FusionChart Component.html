FusionChart Component is an animated flash charting component which offers many types of 2D/3D charts including line, area, bar, column, pie, doughnut (donut), combination, scatter, bubble, scroll charts etc, which make use of Fusion Chart for Flex charting framework.
 

There are 59 different types of charts. The functional and cosmetic aspects of each chart can be extensively customized using the XML api exposed by FusionCharts. You can get more detailed information of XML api at online documentation page of FusionCharts.

Available Properties

name: Name of the component. Default is Timeline.
title (Optional): Timeline component title
type (Required): Type of the chart
url (Required): The default URL to be used for requests to the server to retrieve the data.
pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.
timeout (Optional): The time interval in seconds to wait the server request completes successfully before aborting.

Available Actions

Clone: Creates a copy of the component with same properties
Delete: Deletes the component

Available Events

onItemClick: Event fired when a chart item (slice, bar, column, etc.) is clicked. Available parameters:

  • params.data: JavaScript object representing series data.
