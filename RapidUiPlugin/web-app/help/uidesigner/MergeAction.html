Action Configuration Dialog

  1. Start with naming your action
  2. Set the type to "Merge"
  3. Enter a condition which specifies when this action will be active (???)
  4. Select trigger type as either "menu" or "event". You will be able to specify one or more triggers for the action.
  5. Select triggering events or menus
    1. if trigger type is "event":
      1. select component and event pairs and click on "Add Event" (Available parameters with the fired event is shown above the Triggering Events list, in a blue box)
    2. if trigger type is "menu":
      1. select Component and Menu Item pairs and click on "Add Menu Item" (Available parameters with the selected menu is shown above the Triggering Menu Items list, in a blue box)
  6. Enter the URL of the script to be invoked (start with scripts folder, i.e. scripts/acknowledge.groovy)
  7. Timeout specifies, in seconds, when the call to the server will be considered to have timed out.
  8. ...

Available Actions

Add RequestParameter: This is where the key/value pairs to be passed into the script called is specified. Multiple parameters can be added.
Edit: Edits the properties on the action configuration dialog
Delete: Deletes the action

Request Parameter

Available Properties

key: parameter name as expected by the called script
value: value of the parameter

Available Actions

Delete: Deletes the request parameter
