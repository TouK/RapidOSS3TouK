Defines row color attributes.

Available Properties

  • color {Required): Row color to be used. ex: #ffffff
  • visible (Optional): The JavaScript expression evaluated on row data to determine whether the entire row will be colored using this color or not. Default is "true".
    params.data is available to write the expressions.

Additional Properties

  • textColor (Optional): Parameter to determine the text color when a row is highlighted with "color".

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
