What triggers an action is defined by adding one or more trigger nodes to the action. In RapidInsight there are 4 types of events that can trigger an action. These trigger types are:

  • Menu: action will be triggered when a menu item is clicked
  • Component event: action will be triggered when an event supported by a component is fired, such as "rowClicked" event of the SearchGrid
  • Action event: action will be triggered by an event fired by a preceding action, such as the "success" event of the FunctionAction
  • Global event: action will be triggered when a global event is fired, such as serverDown, which indicates communication to the server is lost.

Available Properties

type: Determine the trigger type by selecting what will fire the event: a menu item, a component, an action, or RapidInsight web client (global)
component: When the trigger type is Menu or Component, select which component fires the desired event or contains the desired menu item
triggeringAction: When the trigger type is Action event, select which action fires the desired event
event: Event fired that will trigger the action

Available Actions

Clone: Clones the trigger for creating a similar trigger quickly
Delete: Deletes the trigger
