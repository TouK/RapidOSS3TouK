Defines image attributes in map node.

Available Properties

  • name (Required): Name of the nodeContent property
  • dataKey (Required): The attribute in map node data which the mapping will use to determine the image.
  • mapping (Required): Map which defines the image files mapping depending on the possible values of "dataKey" attribute.
  • x (Required): Sets how far the left edge of the image is to the left edge of the map node.
  • y (Required): Sets how far the top edge of the image is to the top edge of the map node.
  • width (Required): Width of the image.
  • height (Required): Height of the image.

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
