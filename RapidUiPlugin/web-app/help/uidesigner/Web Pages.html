We will start by defining a web page and work our way down hierarchically.

On any node of the designer tree, you can invoke actions by Click on its context menu (triangle icon to its right)
On any node of the designer tree, available parameters will be displayed in Property Editor pane. Additional properties can be added to the properties list by selecting it from the drop down list for Add field, at the bottom of the Property Editor Pane.

"Web pages" is the root node of the designer tree.

Available Actions on a Web Pages node.

Add New Web Page: Creates a new web page.

The new web page is created with the name "Web Page" under Web Pages node.
