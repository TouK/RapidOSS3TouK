Defines a timeline band.

Available Properties

  • width (Required): The percentage of the band in the whole timeline. ex:50
  • intervalPixels (Required): How many pixel each date unit will span. ex: 100
  • intervalUnit (Optional): Which date unit that the band will display. Possible values are "millisecond", "second", "minute", "hour", "day", "week", "month", "year", "decade", "century", "millennium", "epoch" and "era". Default is "day".
  • showText (Optional): Parameter whether to display event text or not. Default is "false".
  • highlight (Optional): Parameter to display the band whether highlighted or not. Default is "false".

Additional Properties

  • trackHeight (Optional): Track height in "em" unit. See SIMILE Timeline documentation for details.
  • trackGap (Optional): Track gap in "em" unit. See SIMILE Timeline documentation for details.
  • syncWith (Optional): By default all timeline bands scroll independently. The parameter specifies the band index that the current band scroll in synchrony.
  • layoutWith (Optional): The index of the band that the current band layout in synchrony (Events are at the same tracks).
  • date (Optional): Parameter to make sure the timeline starts out showing the events immediately without requiring the user to pan first. Band is positioned at the specified date.
  • textWidth (Optional): The width of the event text in pixels. Default is 200.

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
