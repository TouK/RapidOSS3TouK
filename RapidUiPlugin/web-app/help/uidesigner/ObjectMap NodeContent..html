Defines a node content attributes in map node.

Available Properties

  • name (Required): Name of the nodeContent property
  • dataKey (Required): The attribute in map node data which the mapping will use to determine the node content.
  • x (Required): Sets how far the left edge of the node content is to the left edge of the map node.
  • y (Required): Sets how far the top edge of the node content is to the top edge of the map node.
  • width (Required): Width of the node content.
  • height (Required): Height of the node content.

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
