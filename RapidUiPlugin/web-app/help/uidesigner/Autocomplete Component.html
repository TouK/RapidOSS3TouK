Autocomplete component provides the functionality for text-entry suggestion and completion as the (s)he types in the edit box.

Available Properties

name (Required): Name of the component. Default is Autocomplete
title (Optional): Autocomplete title.
url (Required): The default URL to be used for requests to the server to retrieve the data.
contentPath (Required): The node names of AJAX response which will be used as suggestion data. ex: "Suggestion" for the sample data below.
suggestionAttribute (Required): The attribute of suggestion data which will be displayed in suggestion box. ex" "Name" for the sample data below.

Below is the sample Ajax response in Xml format to a url call:

<Suggestions>
    <Suggestion Name="device1"></Suggestion>
    <Suggestion Name="device2"></Suggestion>
    <Suggestion Name="device3"></Suggestion>
    <Suggestion Name="device4"></Suggestion>
</Suggestions>

Additional Properties

submitButtonLabel (Optional): Submit button text. Default is "Search".
cacheSize (Optional): Max number of data stored in client side cache. Default is 0.
animated (Optional): Parameter to display suggestion box whether animated or not. Default is "false".
timeout (Optional): The time interval in seconds to wait the server request completes successfully before aborting.

Available Actions

Clone: Creates a copy of the component with same properties
Delete: Deletes the component

Available Events

onSubmit: Event fired when a suggestion is selected or "search" button is clicked. Available parameters:

  • params.query: Value in search input field.
