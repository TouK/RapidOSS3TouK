FlexPieChart is a UI component designed to represent data in pie chart.

Available Properties

name: Name of the component. Default is FlexPieChart.
title (Optional): FlexPieChart title
url (Required): The default URL to be used for requests to the server to retrieve the data.
rootTag (Required): The root node name of AJAX response which FlexPieChart takes as starting point to get its data. ex: "chart" for the sample data below
pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.
timeout (Optional): The time interval in seconds to wait the server request completes successfully before aborting.

Below is the sample Ajax response in Xml format to a url call:

<chart>
	<set label='Critical' value='1'  color='0xff0000'/>
	<set label='Major' value='4' color='0xff8c00'/>
	<set label='Minor' value='1' color='0xffff00'/>
	<set label='Warning' value='2' color='0x00ffff'/>
	<set label='Indeterminate' value='3' color='0xac6bac'/>
	<set label='Normal' value='2' color='0x00b200'/>
</chart>
Note that in your response, you should have label, value and color tags.

Available Actions

Clone: Creates a copy of the component with same properties
Delete: Deletes the component
