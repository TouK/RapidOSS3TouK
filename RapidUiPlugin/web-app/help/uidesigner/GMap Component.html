GMap is a UI component which integrates Google Maps with RapidInsight UI. Currently location markers, lines between specified coordinates and icons are supported.

Available Properties

name (Required): Name of the component. Default is GMap.
title (Optional): GMap component title
url (Required): The default URL to be used for requests to the server to retrieve the data.
pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.

Below is the sample Ajax response in Xml format to a url call:

<Locations>
    <Location address='Greater Rochester International Airport' lat='43.1279950' lng='-77.6660980'
              tooltip='&lt;b&gt;Greater Rochester International Airport (2 devices)&lt;/b&gt;'
              marker='http://www.mapbuilder.net/img/icons/marker_34_orange.png' NodeType='Container' Name='CorpVPN1'/>

    <Location address='Logan International Airport' lat='42.3613980' lng='-71.0135050' tooltip='&lt;b&gt;Logan International Airport (7 devices)&lt;/b&gt;'
              marker='http://www.mapbuilder.net/img/icons/marker_34_red.png' NodeType='Container' Name='CorpVPN1'/>
    <Location address='John F Kennedy International Airport' lat='40.6424805' lng='-73.7880710'
              tooltip='&lt;b&gt;John F Kennedy International Airport (6 devices)&lt;/b&gt;'
              marker='http://www.mapbuilder.net/img/icons/marker_34_red.png' NodeType='Container' Name='CorpVPN1'/>
    <Line startLat='43.1279950' startLng='-77.6660980' endLat='42.3613980' endLng='-71.0135050' color='#00ff00'/>

    <Icon lat='42.1279950' lng='-76.6660980' src='images/rapidjs/component/states/red.png' width='16' height='16'/>
</Locations>

Additional Properties

locationTagName (Optional): The node names of AJAX response which will be used as location data. Default is "Location".
lineTagName (Optional): The node names of AJAX response which will be used as line data. Default is "Line".
iconTagName (Optional): The node names of AJAX response which will be used as icon data. Default is "Icon".
lineSize(Optional): Pixel width of the lines. Default is 5.
defaultIconWidth(Optional): Default width of icons in pixels if not specified in AJAX response. Default is 32.
defaultIconHeight(Optional): Default height of icons in pixels if not specified in AJAX response. Default is 32.
timeout (Optional): The time interval in seconds to wait the server request completes successfully before aborting. Default is 30.

Available Actions

Clone: Creates a copy of the component with the same properties
Delete: Deletes the component

Available Events

onMarkerClick: Event fired when a marker is clicked. Available parameters:

  • params.data: JavaScript object representing location data.

onLineClick: Event fired when a line is clicked. Available parameters:

  • params.data: JavaScript object representing line data.

onIconClick: Event fired when a icon is clicked. Available parameters:

  • params.data: JavaScript object representing icon data.
