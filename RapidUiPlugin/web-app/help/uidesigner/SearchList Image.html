Defines image attributes.

Available Parameters

  • src (Required): Image url.
  • visible (Optional): The JavaScript expression evaluated on row data to determine whether the image is to be displayed or not. Default is "true".
params.data is available to write the expressions.

Additional Parameters

  • align (Optional): Sets the starting position of a image. Available values are "left", 'right" and "center". Default is "left".

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
