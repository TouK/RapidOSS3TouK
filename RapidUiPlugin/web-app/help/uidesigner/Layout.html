Available Properties

type: Pre-configured layouts which will automatically add the necessary panes as children nodes. The position of panes with respect to each other will be displayed at the Layout Preview area (upper right quadrant).

Instead of selecting a type, you can manually add layout panes.

Center pane is mandatory and cannot be deleted

Available Actions

Add Top: Adds a new top pane to the layout (above center)
Add Bottom: Adds a new bottom pane to the layout (below center)
Add Left: Adds a new left pane to the layout (left of center)
Add Right: Adds a new right pane to the layout (right of center)
