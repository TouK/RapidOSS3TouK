Available Properties

Name: Name of the tab to be used in the URL. A gsp file with the same name will be generated and accessed with the following syntax (after it is saved and generated):

http://<RapidInsight host>:<port>/RapidSuite/<containing web page name>/<tab name>.gsp

Title: Title of the tab

A newly created tab has the default name "Tab" but you can change it to any name (use letters only with no spaces) in the property editor pane just beneath the tree. Just click on the value field and type the new value for name. When the name is changed, it will be reflected in the tree.

Each tab node contains the following child nodes:

  • Layout (containing the Center region by default): Determines where components should be placed on the tab
  • Components: Which components are to be placed on the tab
  • Dialogs: Any dialogs to be configured on the tab
  • Actions: Stitching together components
In order for a tab to be complete, at least one component and its layout have to be defined.

Additional Properties

contentFile: is a gsp file with some code which can not be supported by UI Designer. This code can be anything that can go into an html file such as a javascript function, or a css code, etc. They will be referenced from UI Designer; for example, you can refer to a javascript function in this file for request parameter value for actions. The file location should be with respect to RS_HOME/RapidSuite/web-app folder.

Available Actions on a Tab node.

Delete: Deletes the tab and all of its contents.
