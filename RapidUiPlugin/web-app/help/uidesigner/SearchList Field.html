Available Parameters

  • fields (Required): Property list separated by comma, that will be displayed if the expression evaluates to true. ex: elementName,createdAt,acknowledged
  • exp (Required): The JavaScript expression evaluated on row data to determine whether the property list is valid or not. Default is true. ex: params.data.rsAlias == 'NetcoolEvent'
params.data is available to write the expressions.

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
