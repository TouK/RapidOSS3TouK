Available Properties

component (Required): Name of the component which will be displayed in this dialog window. It is a drop down list box which displays the components already defined
title (Optional): Dialog window title
width (Required): Width of the dialog window
height (Required): Height of the dialog window

Additional Properties

x (Optional): X location of dialog window when it pops up
y (Optional): Y location of dialog window when it pops up
maxHeight (Optional): Maximum height of dialog window when it is resized
minHeight (Optional): Minimum height of dialog window when it is resized
maxWidth (Optional): Maximum width of dialog window when it is resized
minWidth (Optional): Minimum width of dialog window when it is resized
resizable (Optional): Option to allow resizing. Default is true. If it is set to true, maximum and minimum height and width values should be provided.

Available Actions

Clone: Creates a copy of the dialog with the same properties
Delete: Deletes the dialog
