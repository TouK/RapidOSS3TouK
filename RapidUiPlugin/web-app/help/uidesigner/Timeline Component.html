Timeline is a UI component designed to visualize time-based events.

Available Properties

name: Name of the component. Default is Timeline.
url (Required): The default URL to be used for requests to the server to retrieve the data.
pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.
title (Optional): Timeline component title

Below is the sample Ajax response in Xml format to a url call:

<data>

        NOTIFICATION-Switch_ercaswroc_Unresponsive
    </event>
    <event title='ercaswjfk ercaswjfkLinkMonJavaAdmin' start='Jan 10 2009 12:02:28 GMT' end='Jan 12 2009 23:02:28 GMT' isDuration='true'>
        ercaswjfkLinkMonJavaAdmin
    </event>
    <event title='eraaswpit NOTIFICATION-Switch_eraaswpit_Unresponsive' start='Jan 12 2009 22:02:28 GMT' end='Jan 12 2009 23:02:28 GMT' isDuration='true'>
        NOTIFICATION-Switch_eraaswpit_Unresponsive
    </event>
    <event title='ercaswjfk ercaswjfkMachineMonJavaAdmin' start='Jan 10 2009 17:02:28 GMT' end='Jan 12 2009 22:02:28 GMT' isDuration='true'>
        ercaswjfkMachineMonJavaAdmin
    </event>
</data>

Available Actions

Add Band: Creates a timeline band.
Clone: Creates a copy of the component with same properties
Delete: Deletes the component

Available Events

onTooltipClick: Event fired when a tooltip of an event is clicked. Available parameters:

  • params.data: JavaScript object representing event data.
