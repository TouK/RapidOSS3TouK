Available Properties

Name: Name of the web page to be used in the URL. A gsp file with the same name will be generated and accessed with the following syntax (after it is saved and generated):

http://<RapidInsight host>:<port>/RapidSuite/<web page name>.gsp

A newly created web page has the default name "Web Page" but you can change it to any name (use letters only with no spaces) in the property editor pane just beneath the tree. Just click on the value field and type the new value for name. When the name is changed, it will be reflected in the tree.

Each web page will be created with the Tabs child node, so that one or more tabs can be added to the web page.
