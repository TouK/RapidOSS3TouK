> PieChart is a UI component designed to represent data in pie chart.

Available Properties

name: Name of the component. Default is PieChart.
title (Optional): PieChart title
url (Required): The default URL to be used for requests to the server to retrieve the data.
contentPath (Required): The node names of AJAX response which will be used as row data. ex: "set" for the sample data below
categoryField (Required): The attribute in the data to define x series. ex: "label" for the sample data below
dataField (Required): The attribute in the data to define y series. ex: "value" for the sample data below
legend (Optional): The position of the legend. Possible values are "top", "right", "bottom", "left" and "none". Default is "none".
pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.

Below is the sample Ajax response in Xml format to a url call:

<chart>
	<set label='Critical' value='1' />
	<set label='Major' value='4' />
	<set label='Minor' value='1'/>
	<set label='Warning' value='2'/>
	<set label='Indeterminate' value='3'/>
	<set label='Normal' value='2'/>
</chart>

Additional Properties

colors (Optional): The color definition list of the slices. The order of the color data will match the order of categories in the result set. ex: 0xff0000,0xff7514,0xddc700,0x2dbfcd,0xac6bac,0x00ff00

Available Actions

Clone: Creates a copy of the component with same properties
Delete: Deletes the component
