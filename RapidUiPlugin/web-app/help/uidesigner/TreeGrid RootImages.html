TreeGrid RootImages node is the parent node for all the root menu images defined for the selected TreeGrid component.

Each defined image has an expression that determines when this image can be displayed. Therefore, at any given time, only one of the expressions should be evaluated as true and its corresponding image will be displayed as the row image.

It has no properties of its own.

Available Actions on an RootImages node

Add RootImage: Creates a new image node with the default name "RootImage".
