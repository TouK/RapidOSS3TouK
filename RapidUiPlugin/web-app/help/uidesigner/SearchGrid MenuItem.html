Defines a menu item attributes.

Available Parameters

  • name (Required): The name of the menu item
  • label (Required): The label of the menu item.
  • visible (Optional): The JavaScript expression evaluated on row data to determine whether the item is displayed or not. Default is "true". ex: params.data.acknowledged

Available Actions

Add MenuItem: Adds submenu items

Clone: Creates a copy of the node with same properties
Delete: Deletes the node

Note that the actions which will be executed when a menu item is selected are defined in Actions node for the tab.
