RowColors node is the parent node for all row colors defined for the selected SearchGrid component.

Each defined row color has an expression that determines when the entire row will be colored in that color. Therefore, at any given time, only one of the expressions should be evaluated as true and its corresponding color will be used to highlight the entire row.

It has no properties of its own.

Available Actions on a RowColors node

Add RowColor: Creates a new row color node with the default name "RowColor".
