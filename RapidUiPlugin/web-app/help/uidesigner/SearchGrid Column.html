Defines column attributes.

Available Properties

  • type (Required): Column type. It can be either a text column or a link column. A link column allows the user to move the focus to another tab/dialog window etc. Default is text.
    The action for the link should be defined in Actions node for the tab
  • attributeName (Required): The data node attribute which will be shown in the column. This attribute comes with the Ajax response.
  • colLabel (Required): Title of the column.
  • width (Required): Width of the column.

Additional Properties

  • sortBy (Optional): Parameter to render SearchGrid whether sorted according to this column or not. Default is false.
  • sortOrder (Optional): The order of the sort when 'sortBy" property is "true". Default is "asc".

Available Actions

Clone: Creates a copy of the node with same properties
Delete: Deletes the node
