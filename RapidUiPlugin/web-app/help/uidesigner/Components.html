Components node is the parent node for all components defined for the selected tab.

It has no properties of its own.

Available Actions

This is where which of the supported UI components will be on the tab is declared. Each added component will map to one of the layout panes. If 3 components are added, the layout should have 3 panes to accommodate these 3 components.

  • Add SearchGrid: Adds a SearchGrid component: Search results are displayed in a grid
  • Add SearchList: Adds a SearchList component: Search results are displayed in rows of adjustable height
  • Add TreeGrid: Adds a TreeGrid component: Tree with columns for displaying additional data in one or more cells for each node
  • Add Html: Adds an HTML component
  • Add FusionChart: Adds a Fusion chart component
  • Add FlexPieChart: Adds a Flex pie chart component
  • Add FlexLineChart: Adds a Flex line chart component
  • Add PieChart: Adds a pie chart component
  • Add ObjectMap: Adds a map component to display objects in a diagram of connected nodes
  • Add Autocomplete: Adds an autocomplete component
  • Add AudioPlayer: Adds an AudioPlayer component
  • Add GMap: Adds a Google map component
  • Add Timeline: Adds a timeline component
