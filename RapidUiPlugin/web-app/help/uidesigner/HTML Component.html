Html is a UI component designed to display custom html content.

Available Properties

name (Required): Name of the component. Default is Html.
title (Optional): Html component title

Additional Properties

pollingInterval (Optional): Time delay between two server requests. Default is 0 which means there is no polling.
iframe (Optional): Parameter to display the content in an iframe or embed it to Html component's body. Default is"false".

Available Actions

Clone: Creates a copy of the component with the same properties
Delete: Deletes the component
